Little Lemon

Restaurant & API

Welcome to the Little Lemon Restaurant API. Use the endpoints below to interact with our menu and booking system.

API Endpoints

  • /api/menu/ — Menu items (CRUD)
  • /api/bookings/ — Bookings (CRUD)
  • /api-token-auth/ — Obtain auth token
  • /auth/users/ — User registration
  • /auth/token/login/ — Djoser login
  • /auth/token/logout/ — Djoser logout
